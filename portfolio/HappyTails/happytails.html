Happy Tails: Adopt a Shelter Pet

Learning social media management.

I created a social media campaign for an organization called Happy Tails, which specialized in encouraging people to adopt animals from local shelters. For this campaign, I learned how to develop social media audits to learn the current state of a business’s social media presence, created unique graphics, and learned how to use Hootesuite to keep track of social media analytics and engagement. The two biggest challenge that I faced during this project was learning how to use Hootesuite’s dashboard and developing a Wordpress site, since this was my first time using both of these programs.
